# USER GUIDE

## Quick Start

### Installation via MSI
1. Download the latest MSI installer from the official repository.
2. Double-click the downloaded MSI file to begin the installation process.
3. Follow the prompts in the installation wizard to install the application.

### Launching the Application
After installation, you can launch the application by:
1. Clicking on the 'Start' menu.
2. Searching for 'AI Microscope'.
3. Clicking on the application icon to open it.

### Dynamic Loading of AI Model
Upon launching, the application dynamically loads the AI model from the following directory: 
```
src/dmbaimicroscope/model/
```
Ensure that the required model files are located in this directory for optimal performance.

## Clinical Workflow
1. Prepare the microscope and specimen as per standard operating procedures.
2. Open the AI Microscope application and load the required AI model.
3. Follow the on-screen instructions to process the specimen images.
4. Review results and save necessary data.

## Specimen Handling
- Store specimens in a suitable environment to maintain integrity.
- Follow specific handling guidelines for each specimen type to ensure accurate imaging results.

## Results Interpretation
- Analyze the results using the application’s built-in tools.
- Utilize provided interpretation guides to assist in understanding the AI's assessments.

## Troubleshooting
- If the application fails to launch, check the installation and ensure all dependencies are installed.
- For issues with model loading, verify that the model files are correctly placed in `src/dmbaimicroscope/model/`. 
- Consult the user manual or contact support for further assistance.
